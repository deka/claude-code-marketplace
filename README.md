# deka-plugins — Claude Code Marketplace

Personal plugin marketplace for [Claude Code](https://claude.com/code).

## Installation

To add this marketplace from a public GitHub repository:

```
/plugin marketplace add deka/claude-code-marketplace
```

Or using the shortcut:

```
/plugin market add deka/claude-code-marketplace
```

## Available Plugins

### marp-tools-plugin

Complete [Marp](https://marp.app/) presentation toolkit — create slides, design custom themes, fix visual issues, export to PDF/PNG/PPTX/HTML, and set up automated build pipelines.

| Skill | Description |
|-------|-------------|
| **marp-markdown** | Create and edit Marp slides — content, structure, frontmatter, directives |
| **marp-knowledge** | Marp syntax reference — directives, themes, images, math/code |
| **marp-theme-creator** | Design custom themes from scratch — SCSS/CSS, color palettes, branding, slide classes |
| **marp-visual-fix** | Diagnose and fix display/layout problems — overflow, images, broken layouts, PDF rendering |
| **marp-export** | Export to PDF, PNG, PPTX, or HTML for sharing, printing, or web publishing |
| **marp-theme-builder** | Automate theme builds — SCSS compilation, Makefile, Git hooks, GitHub Actions CI |
