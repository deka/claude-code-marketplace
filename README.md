# deka-plugins — Claude Code Marketplace

Personal plugin marketplace for [Claude Code](https://claude.com/code).

## Installation

To add this marketplace from a public GitHub repository:

```
/plugin marketplace add deka/claude-code-marketplace
```

Or using the shortcut:

```
/plugin market add deka/claude-code-marketplace
```

## Available Plugins

### marp-tools-plugin

Skill set for creating and managing [Marp](https://marp.app/) presentations.

| Skill | Description |
|-------|-------------|
| **marp-markdown** | Create and edit slides in Markdown (content, frontmatter, directives) |
| **marp-knowledge** | Marp knowledge base (syntax, directives, themes, images, etc.) |
| **marp-theme-creator** | Create custom themes (SCSS/CSS, palette, typography) |
| **marp-visual-fix** | Fix display and layout issues |
| **marp-export** | Export to PDF, PNG, PPTX, HTML |
| **marp-theme-builder** | Set up build pipelines (SCSS compilation, Makefile, CI, pre-commit) |
